# VoxelCraft Prompt Kit

Reference research and specification files for building VoxelCraft.

## File Index

- `00-master-prompt.md` — THE PROMPT. Mission, stack, structure, features, acceptance criteria.
- `01-research-blocks.md` — All ~800 blocks with hardness, light, drops, properties.
- `02-research-items.md` — All ~1,580 items: tools, weapons, armor, food, materials.
- `03-research-mobs.md` — All mobs with HP, damage, AI behavior, goals, pathfinding.
- `04-research-biomes.md` — All ~60 biomes, multi-noise generation algorithm.
- `05-research-dimensions.md` — Overworld, Nether, End: dimension properties, portal mechanics.
- `06-research-mechanics.md` — Player movement, hunger, combat, redstone, enchanting, brewing.
- `07-research-recipes.md` — All ~880 crafting/smelting/blasting/smithing recipes.
- `08-research-structures.md` — All structures: villages, fortresses, end cities, trial chambers.
- `09-research-settings.md` — Every vanilla options menu setting.
- `10-architecture.md` — Tech architecture: Three.js, chunk system, greedy meshing, mob AI.
- `11-features.md` — Detailed feature specs with acceptance criteria.
- `12-deployment.md` — Vercel deployment guide.
